## 플러그인 상세 설명

### 자체 제작 플러그인
*   **AutoRestart**: 지정된 시간에 서버를 자동으로 재시작하여 안정적인 상태를 유지합니다.
*   **MenuBook**: 인벤토리 UI를 통해 서버의 주요 기능을 안내하고 편리한 이동을 돕습니다.
